## 二、各数据集详细分析

### 2.1 GSE226211 — 皮层刺伤后胶质细胞共享炎症信号

**核心价值：与你造模完全一致的 scRNA-seq + 空间转录组**

#### 实验设计

- 19G细刀插入灰质，沿前后轴移动1mm（从 Bregma -1.2 到 -2.2mm）
- 时间点：Intact / 3dpi / 5dpi
- 包含通路抑制剂组（Cxcr3拮抗剂 NBI 74330 + Tlr1/2抑制剂 CU CPT 22）

#### 小胶质细胞亚群（8个亚簇）

| 亚簇                    | 状态             | 特征                       |
| ----------------------- | ---------------- | -------------------------- |
| MG1, MG2, MG3, MG5, MG8 | 稳态             | Tmem119+, P2ry12+          |
| **MG4, MG6, MG7** | **反应性** | Aif1 高, Tmem119/P2ry12 低 |

#### 核心发现

- **241个共享上调基因**横跨反应性胶质亚群（MG4, AG5, OPC2）
- I型干扰素信号是共享炎症核心：Ifitm3, Ifit3, Bst2, Isg15, Irf7, Ifit1
- 趋化信号：CXCL10（CXCR3配体）
- 通路逻辑：TLR2 → IRF7 → I型IFN → CXCL10转录

#### 局限性

- **无脂质代谢分析**（原文未涉及）
- **时间点仅到5dpi**，无法覆盖你的7d峰值和14d消退
- 需自行重分析反应性小胶质中的LDAM基因特征

#### 可用数据

- scRNA-seq: Cell Ranger输出（barcodes/features/matrix）, 606 MB
- Visium: Space Ranger输出 + 组织图像, 95 MB
- 无预计算的细胞注释文件（需自行聚类）

### 2.2 GSE163691 — 皮层刺伤后小胶质时间序列转录组（⭐ 最匹配你的时间窗口）

**核心价值：D1/D4/D7/D14 四个时间点的纯化小胶质 RNA-seq，完美覆盖脂滴动态窗口**

#### 实验设计

- 皮层刺伤造模（与你一致）
- Cx3cr1CreER 小鼠 → FACS 分选小胶质细胞
- 损伤侧 vs 对侧对照
- **雌雄均有**（每组2个生物学重复）

#### 数据结构（32个样本）

```
每个时间点 (D1/D4/D7/D14):
  ├── Male_Sham × 2 replicates
  ├── Male_Injury × 2 replicates
  ├── Female_Sham × 2 replicates
  └── Female_Injury × 2 replicates
= 8 samples × 4 timepoints = 32 samples
```

#### 核心发现

- 小胶质呈现**时间特异性分子特征**
- 存在显著的**性别差异**（与神经退行性疾病性别偏好相关）

#### 对你课题的关键价值

1. **D7 = 脂滴峰值期**，D14 = 脂滴消退期 → 直接匹配你的观察
2. 可查询 LDAM 标志基因（Lpl, Apoe, Cd36, Fabp5, Plin2/3, Acsl1）的时间轨迹
3. 可查询 DAM 标志基因（Trem2, Tyrobp, Cst7, Spp1, Itgax, Axl, Lgals3）
4. 性别分层数据 → LDAM 在衰老中有性别差异（Marschallinger 2020）

#### 可用数据（非常完整）

| 文件                                 | 内容                           |
| ------------------------------------ | ------------------------------ |
| `GSE163691_ReadCount.txt.gz`       | 原始读数计数矩阵               |
| `GSE163691_NormalizedCount.txt.gz` | 标准化计数                     |
| `GSE163691_TPM.txt.gz`             | TPM表达值                      |
| `*_Diff.txt.gz` × 12              | 各时间点差异表达（合并/男/女） |

### 2.3 GSE111452 — ⚠️ 与预期不符

**重要更正**：此数据集实际为：

- ❌ **大鼠**（非小鼠）
- ❌ **FPI 液压冲击**（非 CCI 控制性皮层撞击）
- ❌ **Agilent 微阵列**（非 RNA-seq）
- 时间点：24h / 2周 / 3月 / 6月 / 12月（**无7天时间点**）

#### 仍有参考价值

- 长期时间序列（至12个月）揭示慢性神经炎症
- NF-κB、补体（C2/C3）、TLR4、**IL-6 信号**持续激活
- 84只大鼠、113个样本，统计效力强
- 但**无脂质代谢发现**，**无小胶质特异分析**

### 2.4 GSE180862 — 唯一包含外周血的单细胞数据集

**核心价值：脑 + 血 匹配的单细胞转录组，支撑"脑-外周串扰"假说**

#### 实验设计

- 小鼠 C57BL/6J，轻度 FPI (1.4-1.6 atm)
- 组织：**前额皮层 + 海马 + 外周血白细胞**
- 时间点：24h + 7d
- 包含 Humanin 治疗组

#### 细胞类型

| 组织            | 主要细胞类型                                          |
| --------------- | ----------------------------------------------------- |
| 外周血 (8簇)    | CD8+ T, CD4+ T, Ly6c+ 单核, B细胞等                   |
| 前额皮层 (13簇) | 星形胶质,**活化小胶质**, 小胶质, 少突, 内皮等   |
| 海马 (17簇)     | 星形胶质,**活化小胶质**, 小胶质, 多种神经元亚型 |

#### 关于小胶质的发现

- 区分了"小胶质"和"活化小胶质"两个明确簇
- **24h**：海马小胶质强烈激活（Tnf, Il1b, Ly86上调）
- **7d**：皮层活化小胶质呈延迟反应
- 意外发现：**星形胶质**（而非小胶质）在所有时间点和区域中转录扰动最大

#### 脑-外周串扰发现（⭐ 对你假说最重要）

- **mt-Rnr2, mt-Rnr1, Malat1, Rimklb** 在脑和血液中**同步扰动**
- 外周免疫反应在 7d 时仍持续，而 CNS 免疫反应已消退
- CD8+ T 细胞和 Ly6c+ 单核细胞在急性和亚急性期均有时间特异性变化

#### 局限性

- **无脂质代谢通路富集**
- **未报道 IL-6/JAK-STAT 通路**（炎症主要围绕 TNF/IL-1β）
- 无肝脏组织数据

### 2.5 OMIX011437 — 卒中后小胶质胆固醇代谢重编程（⭐ 脂质机制最直接的参考数据集）

**核心价值：目前唯一直接揭示脑损伤后小胶质胆固醇/脂质代谢重编程机制的转录组数据集**

> 来源：Zhao et al., *Nature Metabolism* 2025 (PMID: 40987840)
> 数据：https://ngdc.cncb.ac.cn/omix/release/OMIX011437

#### 实验设计

- 小鼠 C57BL/6J（雄性），MCAO 缺血性卒中模型
- 富集 CD11b+ 小胶质细胞 → Bulk RNA-seq
- 全脑组织 → scRNA-seq (10× Genomics)
- 药物干预：efavirenz（CYP46A1 激活剂，促进胆固醇代谢清除）
- 同时包含：脂质组学（LC-MS/MS）、免疫组化、行为学评估、人类卒中后脑组织验证

#### 数据结构（12个样本）

```
Bulk RNA-seq:
  ├── MCAO-1 × 3 replicates（早期时间点）
  ├── MCAO-2 × 3 replicates（晚期时间点）
  ├── Sham-1 × 3 replicates
  └── Sham-2 × 3 replicates
scRNA-seq: 全脑组织，具体细胞数未公开
```

#### 核心发现

- 卒中后小胶质出现**持续性**（慢性）激活，伴随胆固醇积累
- 鉴定出"**泡沫小胶质**（foamy microglia）"亚群：高表达 *Apoe*, *Lpl*, *Trem2*, *Cd36*, *Abca1*, *Cyp46a1*
- 直接向脑内注射胆固醇可**复现**持续性小胶质激活表型
- **CYP46A1**（胆固醇代谢酶）激活：
  - 减少小胶质脂质负担
  - 抑制慢性神经炎症
  - 改善卒中后功能恢复
- 人类卒中后脑组织中验证了相同的泡沫小胶质特征

#### 与你TBI-脂滴课题的关联（⭐⭐⭐）

| 维度                   | OMIX011437的证据                       | 对你课题的意义                           |
| ---------------------- | -------------------------------------- | ---------------------------------------- |
| **脂质积累机制** | 胆固醇代谢重编程驱动泡沫小胶质         | 提供TBI后脂滴形成的机制类比              |
| **基因标志物**   | Apoe, Lpl, Trem2, Cd36, Abca1 高表达   | 与LDAM特征基因高度重叠                   |
| **慢性化机制**   | 胆固醇积累 → 持续促炎 → 阻碍神经修复 | 支持脂滴/脂质负担在神经修复中的负面作用  |
| **干预靶点**     | CYP46A1激活可清除脂质改善预后          | 提示脂质代谢调控为治疗靶点               |
| **跨病种验证**   | 卒中和TBI均有小胶质脂质积累            | 强化"脂质代谢是脑损伤共同病理机制"的论点 |

#### 局限性

- ❌ **造模为 MCAO 缺血性卒中**，而非 TBI（机制有差异：缺血 vs 创伤）
- ❌ 仅雄性，无法进行性别差异分析
- ❌ **无肝脏数据**，无法分析脑-肝轴
- ⚠️ 数据存储于 NGDC（中国数据库），访问需注册账号
- ⚠️ 聚焦**胆固醇**，对甘油三酯（TG）和磷脂的分析相对有限

#### 可用数据

- Bulk RNA-seq：CD11b+ 富集小胶质，12个样本
- scRNA-seq：全脑单细胞转录组（含小胶质亚群注释）
- 脂质组学数据（文章附表或补充材料）

## 三、数据集优先级排序（针对你的课题）

### 🥇 第一优先：GSE163691

**理由：**

- 皮层刺伤造模完全一致
- D7/D14 时间点完美匹配脂滴动态
- 纯化小胶质 bulk RNA-seq → 信噪比最高
- 已有处理好的 TPM 和差异表达文件
- 性别分层数据

**建议分析：**

1. 下载 TPM 矩阵 → 提取 LDAM/DAM 基因时间轨迹
2. 识别 D7 峰值 + D14 恢复的基因模式
3. 通路富集分析（脂质代谢、吞噬、炎症消退）
4. 性别差异比较

### 🥈 第二优先：GSE226211

**理由：**

- 造模完全一致
- scRNA-seq 提供细胞亚群分辨率
- 空间转录组显示损伤梯度
- 但仅到 5dpi，无法覆盖 D7 和 D14

**建议分析：**

1. 重分析反应性小胶质亚群 (MG4/MG6/MG7) 的 LDAM 特征
2. 空间转录组分析脂质相关基因的损伤梯度
3. 与 GSE163691 的 D4 时间点交叉验证

### 🥉 第三优先：GSE180862

**理由：**

- 唯一有外周血数据的数据集 → 支持脑-外周假说
- 但造模方式不同（FPI vs stab wound）
- 无肝脏数据

**建议分析：**

1. 比较脑 vs 血的脂质代谢基因表达
2. 分析外周免疫细胞中的脂代谢变化
3. 寻找 IL-6 通路在血液中的证据

### 🏅 第四优先：OMIX011437（脂质机制参考）

**理由：**

- 直接研究脑损伤后小胶质**胆固醇代谢重编程**，与你的脂滴课题机制最相关
- 泡沫小胶质基因特征（Apoe/Lpl/Trem2/Cd36）与LDAM高度重叠
- 但造模为卒中（MCAO），不是TBI，需谨慎类比

**建议分析：**

1. 提取泡沫小胶质 marker 基因列表 → 与GSE163691 D7时间点对比
2. 分析 CYP46A1 及胆固醇代谢通路基因在TBI数据集中的表达
3. 作为"脂质代谢调控改善神经修复"的文献证据背书

### ⚠️ 参考使用：GSE111452

**理由：**

- 物种不同（大鼠）、造模不同（FPI）、平台不同（微阵列）
- 长期时间序列可作为慢性炎症参考
- IL-6 信号持续激活的发现支持你的假说方向

## 四、与你课题假说的关联分析

### 4.1 你的核心观察

```
皮层刺伤 TBI → 损伤区脂滴堆积（主要在小胶质细胞）
                → 7d 脂滴最多，14d 完全消失
                → 脑脂质组：含DHA的磷脂和TG显著上调
                → 肝脏：显著脂代谢改变
                → 血清：甘油三酯不饱和度上升
```

### 4.2 你的假说

> **大脑中的不饱和脂肪酸是外周来源的；肝脏对小胶质脂滴形成/消失有影响；肝脏 IL6RA 敲除会改变脑中脂滴，影响 TBI 进程。**

### 4.3 文献支撑程度

#### ✅ 已有充分文献支持的部分

| 观点                                   | 支持文献                                                          | 证据级别 |
| -------------------------------------- | ----------------------------------------------------------------- | -------- |
| TBI后小胶质形成脂滴                    | Marschallinger 2020 (PMID:31959936); Zambusi 2022 (PMID:36424432) | 强       |
| LDAM是功能障碍状态（吞噬缺陷、促炎）   | Marschallinger 2020                                               | 强       |
| 脂滴含胆固醇酯和TG，来源于髓鞘碎片吞噬 | Mehrabani-Tabari 2025 (PMID:41282218)                             | 中-强    |
| FASN K673超乙酰化驱动TBI后脂滴形成     | Zhang 2026 (PMID:41412037)                                        | 中       |
| GRN/progranulin介导脂滴清除            | Zambusi 2022; Marschallinger 2020                                 | 强       |
| TBI后肝脏脂代谢改变                    | Rege 2019 (PMID:31087499)                                         | 中       |
| IL-6介导TBI后肝脏急性期反应            | Yang 2013 (PMID:23172820)                                         | 中       |
| IL-6/GP130信号具有抗脂肪变性作用       | Carbonaro 2023 (PMID:37058568)                                    | 中       |
| 脑DHA通过MFSD2A转运体从外周输入        | Nguyen 2014 (多篇)                                                | 强       |

#### ⚠️ 有间接证据但缺乏直接证明

| 观点                           | 现状                                                                          | 研究空白                               |
| ------------------------------ | ----------------------------------------------------------------------------- | -------------------------------------- |
| TBI脑中DHA磷脂/TG上调          | 多数文献报道DHA磷脂**下降**（而非上升）；你的数据若显示上调可能是新发现 | 需确认你的脂质组数据是否与现有文献矛盾 |
| 肝脏合成的DHA输送到受损大脑    | 肝脏是DHA合成主要器官 + MFSD2A是脑DHA输入通道，但**无直接追踪研究**     | 需同位素标记实验                       |
| 血清TG不饱和度变化反映肝脏供应 | 合理推论但无直接证据                                                          | 你的数据可能是首次报道                 |

#### ❌ 尚无文献支持的关键假说

| 假说                                     | 现状                                                                       |
| ---------------------------------------- | -------------------------------------------------------------------------- |
| **肝脏 IL6RA 敲除影响脑脂滴**      | 无任何相关研究；这是完全新颖的假说                                         |
| **外周来源的脂肪酸进入小胶质脂滴** | 现有证据显示脂滴主要来源于**髓鞘碎片吞噬**（局部来源），而非外周输入 |
| **TBI后肝脏转录组数据**            | GEO中**不存在**任何TBI后肝脏测序数据集                               |

### 4.4 对假说的关键挑战

⚠️ **重要提醒：现有证据倾向于小胶质脂滴的"局部来源"假说**

1. **Mehrabani-Tabari 2025** 明确证明脂滴内容物（胆固醇酯+TG）来源于**髓鞘碎片的吞噬处理**
2. 小胶质通过 TREM2→吞噬髓鞘 → 脂质储存在脂滴中 → 自噬/脂解清除
3. 你需要区分：DHA-TG 是来自吞噬的髓鞘（髓鞘本身富含DHA），还是真的从外周输入

**你假说成立的条件：**

- BBB在TBI后破坏，允许外周脂蛋白进入
- 肝脏特异性改变（如IL-6驱动的TG分泌增加）提供额外脂质
- 这些外周脂质被小胶质摄取并储存为脂滴
- 这需要与髓鞘来源的脂滴**并行**或**叠加**
