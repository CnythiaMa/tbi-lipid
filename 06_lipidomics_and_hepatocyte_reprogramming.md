## 一、脑-血清脂质组联合分析（5dpi）

### 1.1 背景与问题

前序分析（`05_liver_snRNA-seq_analysis.md`）提出肝细胞 `Pla2g4a`（磷脂酶A2）上调（+2.43×），可能将膜磷脂中的 PUFA 释放入血，进而进入脑。本节用脑组织脂质组数据验证该假说，同时探索脑-血清脂质流向。

**数据来源**：

- 脑组织：`data/lipidomics-5dpi/brain/brain-all-lipids.numbers`（1,094 种脂质）
- 血清：`data/lipidomics-5dpi/serum/serum-all-lipids-5dpi_vs_control.xlsx`（1,318 种脂质）

### 1.3 核心发现：溶血磷脂的"镜像模式"

Pla2g4a 水解磷脂的产物为**溶血磷脂（LPC/LPE）+ 游离 PUFA**。如果 Pla2g4a 是向血液释放 PUFA 的主要机制，血清中溶血磷脂应该**升高**。

实际观测结果完全相反：

| 组织                   | LPC 方向          | LPE 方向         | 整体方向 |
| ---------------------- | ----------------- | ---------------- | -------- |
| **脑（5dpi）**   | **up=11，down=0** | **up=6，down=0** | 积累 ↑  |
| **血清（5dpi）** | **up=1，down=10** | **up=0，down=5** | 减少 ↓  |

> **结论**：血清溶血磷脂减少 + 脑溶血磷脂积累 = 教科书式**摄取消耗模式**。脑组织在主动从血清中摄取溶血磷脂，而非血清在输出溶血磷脂。

**Pla2g4a 假说被否定**：肝细胞 Pla2g4a 上调是真实的，但其产物（溶血磷脂）在血清中不升反降，说明：

- Kupffer 细胞基础 Pla2g4a 表达量是肝细胞的 15 倍，且 7dpi 时下降
- 肝细胞中新增的 Pla2g4a 活性不足以驱动血清溶血磷脂升高
- 溶血磷脂被大脑消耗的速率超过了可能的产生增量

### 1.7 修正后的机制模型

```
血清中正常循环的 PUFA-磷脂（LPC(22:6), LPE(20:4)...）
            ↓
   3-5dpi BBB 通透性增加
   + 内皮细胞 Mfsd2a 仍保留约 70% 表达
            ↓
   PUFA-磷脂从血清直接经 BBB 进入脑组织
            ↓
   血清 DHA/AA 磷脂减少 ← 本研究 5dpi 脂质组
   脑组织 LPC/LPE/TG 中 DHA/AA 积累 ← 本研究 5dpi 脂质组
            ↓
   微胶质 Lpl↑ Cd36↑ Fabp5↑ 大量摄取
   Mgll↓ → 脂解受阻 → LDAM 峰值（7dpi）
```

肝脏在此模型中是循环 PUFA-磷脂的**远端供应商**（正常生理功能），而非主动向脑输送脂质的调控节点。

# 二、肝细胞炎症性表型重编程（Identity Shift）

### 2.1 假说来源

在分析肝脏 snRNA-seq 差异表达结果时，观察到 7dpi 肝细胞中出现了一批通常只在 Kupffer 细胞/巨噬细胞中表达的基因。基于此提出假说：**脑外伤后肝细胞的转录特征发生了朝向 Kupffer 细胞方向的偏移**。

---

### 2.2 分析方法：Kupffer Cell Identity Score

#### 第一步：定义特征签名基因集

使用 **one-vs-rest Wilcoxon 检验**，在 **Control 组**肝脏核中分别提取：

- **Kupffer 签名（100 个基因）**：在 Kupffer 细胞中显著高于所有其他细胞类型的基因
  - 筛选条件：Kupffer 中表达比例 > 15%，且 log2FC(Kupffer vs 其余) ≥ 1.0
- **Hepatocyte 签名（100 个基因）**：同理，肝细胞特异性高表达基因

在 Control 组定义签名的原因：**避免用 7dpi 状态的细胞来定义"正常"身份**——如果 7dpi 的 Kupffer 细胞本身已经发生变化，用它定义的签名会引入循环偏差。

#### 第二步：score_genes 打分

使用 `scanpy.tl.score_genes`（改良版 Seurat `AddModuleScore`），对所有肝细胞核计算两个得分。

**算法细节**：

```
对每个细胞核 c，计算 KupfferScore(c)：

1. 计算目标基因集的平均表达量：
   target_mean(c) = mean( expr(c, g) for g in Kupffer_gene_set )

2. 对每个目标基因 g，从与 g 表达水平相近的"表达量 bin"
   中随机抽取 n_ctrl 个对照基因（默认 50 个）
   组成随机对照基因池 control_pool

3. 计算随机对照均值：
   control_mean(c) = mean( expr(c, g') for g' in control_pool )

4. KupfferScore(c) = target_mean(c) − control_mean(c)
```

**为什么要减去对照均值？**

- 不同细胞核的测序深度（UMI 总量）不同，深度高的核几乎所有基因表达量都会偏高
- 如果直接用目标基因均值，测序深度高的核会天然得到更高的分数
- 从表达水平相似的 bin 中随机抽取对照，相当于为每个基因**自适应地估计背景期望**
- 相减后的得分反映的是目标基因集是否**超过背景期望**，与测序深度解耦

**得分解读**：

- `KupfferScore > 0`：该核中 Kupffer 特征基因的表达高于背景期望，表现出 Kupffer 样特征
- `KupfferScore < 0`：低于背景，缺乏 Kupffer 特征
- `IdentityRatio = KupfferScore − HepatocyteScore`：综合双向偏移指标
  - 正值 → 更像 Kupffer
  - 负值 → 更像 Hepatocyte

#### 第三步：两组比较（Wilcoxon rank-sum）

对 control 组和 7dpi 组的肝细胞核得分分布进行双侧 Wilcoxon 检验。

---

### 2.3 结果

#### 得分统计

| 指标                    | Control          | 7dpi                       | Δ               | p 值                          |
| ----------------------- | ---------------- | -------------------------- | ---------------- | ----------------------------- |
| **KupfferScore**  | -0.0723 ± 0.059 | **0.0000 ± 0.087**  | **+0.072** | **p ≈ 0**              |
| HepatocyteScore         | -0.0067 ± 0.116 | -0.0480 ± 0.140           | -0.041           | p = 5.9×10⁻⁹⁴             |
| **IdentityRatio** | -0.0656 ± 0.154 | **+0.0480 ± 0.207** | **+0.114** | **p = 1.1×10⁻²⁸⁸** |

- Control 组肝细胞 KupfferScore 为负（远低于背景），Identity Ratio 也为负（典型肝细胞特征）
- 7dpi 组 KupfferScore 上升至 0（即达到全转录组背景水平），Identity Ratio 翻转为正
- 三项指标均达到**天文学意义上的显著**（>10,000 个核，Wilcoxon 极端显著）

**关键解读**：

- "Kupffer Score 从 -0.072 升至 0.000"看起来变化不大，但在有 8,000+ 个细胞的数据集里，这个变化的 p 值趋近于 0，说明**几乎每一个 7dpi 肝细胞都有轻微但一致的偏移**，而非少数细胞的极端变化
- Identity Ratio 从负翻正是**质的转变**：Control 组肝细胞更像自己（Hepatocyte），7dpi 时变得更像 Kupffer

### 2.5 驱动机制分析

#### 关键转录因子：Spi1（PU.1）上调

`Spi1` 编码的 **PU.1** 是髓系细胞（单核细胞、巨噬细胞、树突细胞）分化的**主转录因子（master TF）**。正常情况下，成熟肝细胞中 PU.1 几乎不表达。7dpi 时 PU.1 在肝细胞中显著上调（+1.12 log2FC），暗示脑外伤触发了肝细胞中髓系基因程序的解锁。

可能的上游调控链：

```
脑外伤 → IL-6/TNF-α/LPS样信号 → 肝细胞 NF-κB / STAT3 激活
    → Spi1(PU.1) 上调
    → 巨噬细胞谱系基因程序开放
    → Csf1r↑, Adgre1↑, Timd4↑, Clec4f↑, Cd163↑...
```

Il6ra 在肝细胞中 7dpi 上调（+0.81），STAT3→PU.1 轴是已报道的机制。

#### Warburg 代谢重编程：Hk2/Hk3 强烈上调

激活的巨噬细胞放弃氧化磷酸化转向糖酵解（"Warburg 效应"），Hk2/Hk3（己糖激酶）是这一代谢转变的限速酶。肝细胞中 Hk2（+1.29）和 Hk3（+1.73）的强烈上调，提示 7dpi 肝细胞不仅在表面标志物上像巨噬细胞，在**能量代谢层面也向激活的炎症细胞转变**。

---

### 2.6 与脂代谢的关联

肝细胞获得的 Kupffer 样功能对脂代谢有直接影响：

| 新获得的功能   | 相关基因               | 脂代谢效应                                             |
| -------------- | ---------------------- | ------------------------------------------------------ |
| 脂质清道夫受体 | Cd36↑, Msr1↑, Mrc1↑ | 肝细胞从循环中摄取更多脂质（包括 ox-LDL、PUFA-脂蛋白） |
| 磷脂酶活性     | Pla2g4a↑（+1.28）     | 膜磷脂裂解，但以肝细胞自身（不是向外释放）为底物       |
| 胆固醇酯化     | Soat1↑（+1.27）       | 游离胆固醇→胆固醇酯（巨噬细胞泡沫化特征）             |
| 溶酶体脂解     | Ctss↑                 | 溶酶体内脂质处理                                       |

这些变化与观察到的**肝细胞 Plin2 下调（脂滴减少）+ Cd36 上调（摄取增加）**一致：脂肪酸被摄取后不以脂滴形式储存，而以炎症激活状态消耗（糖酵解 Hk2/Hk3）。

### 2.7 修正后的假说（完整版）

结合 snRNA-seq、脂质组及文献，修正后的假说如下：

```
皮层刺伤脑外伤（7dpi）
    ↓

    ↓

    ↓

    ↓
```

### 2.8 这是一个潜在的新发现

目前文献中**尚无脑外伤后肝细胞向 Kupffer 表型偏移**的直接报道。相关背景包括：

- 炎症性肝病（NASH、酒精性肝炎）中肝细胞可表达部分巨噬细胞基因
- 体外实验中 IL-6 + TNF-α 处理可在肝细胞中诱导 PU.1
- Kupffer 细胞耗竭后，肝细胞可部分补充巨噬细胞功能（文献中称为"hepatocyte-derived macrophage-like cells"）

**本研究的新颖性**：首次在脑外伤动物模型的肝脏 snRNA-seq 中，系统性地记录了肝细胞向 Kupffer 表型偏移的完整基因表达图谱，并提供了潜在的驱动机制（IL-6/STAT3/PU.1 轴）和脂代谢关联。
